
# Data Cleaning Task (Task 1)

## Objective
Clean and prepare a raw dataset that contains:
- Missing values
- Duplicates
- Inconsistent formats
- Mixed text cases
- Currency symbols
- Unclean dates

## Tools Used
- Python (Pandas)

## How to Run
1. Install dependencies:
   pip install pandas python-dateutil

2. Run cleaning script:
   python scripts/clean_data.py

3. Output is saved in:
   data/cleaned/cleaned_dataset.csv

## Included Files
- data/raw/raw_dataset.csv (messy dataset)
- data/cleaned/cleaned_dataset.csv (cleaned output)
- scripts/clean_data.py (cleaning script)
- summary_of_changes.txt
- README.md
